# 🍲 Recipes Finder Web App

A simple web app to search and explore recipes in real time using a public API.

---

## 🚀 Features

* Search recipes by name or ingredients
* Fetch real-time data using REST API
* Display recipe images and details dynamically

---

## 🛠️ Tech Stack

* HTML
* CSS
* Vanilla JavaScript (Fetch API)

---

## ⚙️ How It Works

* User enters a search query
* Data is fetched from a public recipe API
* Results are displayed dynamically using DOM manipulation
